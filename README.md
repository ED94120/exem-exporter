# exem-exporter
Extraction capteurs EXEM
Ce code ne fonctionne que si les points de mesure sont visibles sur le graphique temps-mesure.
Si la courbe représentée sur le graphique est une courbe lisse, ce code ne fonctionne pas.
La période de temps maximale qui doit figurer sur le graphique est de 7 jours.
Le programme extrait les informations des pop-up.
